## Evaluate Tab

**Question:** "What is REST and why is it used for APIs?"

**Sample answer to evaluate:**  
"REST stands for Representational State Transfer. It's an architectural style for APIs. It uses HTTP methods like GET, POST, PUT, DELETE. It's stateless and cacheable. REST is popular because it's simple and works well with the web."
